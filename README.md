# SANDECH CRM Comercial de Engenharia

CRM técnico-comercial em **Python + Streamlit + SQLite**, desenhado para a rotina consultiva da SANDECH.

## O que este projeto cobre
- Contas / clientes
- Contatos e influência na compra
- Oportunidades com pipeline técnico-comercial
- Propostas e revisões
- Follow-ups / atividades
- Compliance comercial
- Dashboards com valor, valor ponderado, aging, taxa de conversão e motivos de perda
- Biblioteca de apoio (premissas, fora de escopo, padrões de proposta e listas)

## Como rodar
```bash
pip install -r requirements.txt
streamlit run app.py
```
